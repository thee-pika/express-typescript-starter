# Express TypeScript Starter

A clean and scalable starter template for building backend applications using Express.js and TypeScript.

---

## Features

- Express server setup
- TypeScript support
- Environment variable configuration using dotenv
- Nodemon hot reload for development
- Structured and scalable folder architecture
- Centralized error handling
- Production-ready build configuration

---

# Tech Stack

- Node.js
- Express.js
- TypeScript
- Nodemon
- ts-node

---

# Project Setup

## 1. Clone the repository

```bash
git clone https://github.com/<your-username>/<repository-name>.git <project-name>
```

---

## 2. Move into the project directory

```bash
cd <project-name>
```

---

## 3. Install dependencies

Using npm:

```bash
npm install
```

---

## 4. Configure environment variables

Create a `.env` file in the root directory and add:

```env
PORT
```

---

## 5. Start the development server

Using npm:

```bash
npm run dev
```

---

# Available Scripts

## Run development server

```bash
npm dev
```

---

## Build the project

```bash
npm build
```

---

## Start production server

```bash
npm start
```
